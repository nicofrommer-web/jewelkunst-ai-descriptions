# Jewelkunst AI Descriptions

KI-gestuetzte Textbasis fuer Jewelkunst Produktbeschreibungen, SEO-Texte und Marketing-Copy.

## Zweck

Dieses Repository sammelt Strukturen und Prompts fuer:

- Produktbeschreibungen
- SEO-Titel
- Meta Descriptions
- Social Captions
- E-Mail-Texte
- Markenstorytelling

## Tonalitaet

Ruhig, hochwertig, warm, poetisch und bedeutungsvoll. Keine aggressive Verkaufssprache.

## Beispielstruktur

```text
Produktname
Material
Farbe
Laenge/Groesse
Emotionale Bedeutung
SEO Keyword
Kurzbeschreibung
Langbeschreibung
Meta Description
```

## Marke

Jewelkunst steht fuer Schmuck mit leiser Eleganz und persoenlicher Bedeutung.
