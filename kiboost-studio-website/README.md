# KI Boost Studio – Landingpage

Öffentliche Verkaufs-Landingpage für **KI Boost Studio** (KI Sichtbarkeits-Paket für Kosmetikstudios).

Ergänzt das interne Verkaufs-Cockpit (CRM) um eine kundenseitige Seite zum Teilen/Verlinken (Instagram-Bio, DMs, E-Mail-Signatur).

## Inhalt

- Hero mit Kernversprechen und CTA
- Problem-Sektion (typische Pain Points von Kosmetikstudios)
- Angebot: KI Sichtbarkeits-Paket (499 €/Monat) + kostenlose Potenzialanalyse als Lead-Magnet
- Ablauf/Onboarding in 4 Schritten
- FAQ mit den gängigen Einwänden
- Kontaktformular (öffnet vorausgefüllte E-Mail, kein Backend nötig)

## Nutzung

Einzelne, eigenständige `index.html` – keine Build-Schritte nötig. Einfach im Browser öffnen oder auf einem beliebigen statischen Hoster (GitHub Pages, Netlify, Vercel …) deployen.

## Vor dem Live-Schalten anpassen

- `kontakt@kiboost-studio.de` und `@kiboost.studio` durch echte Kontaktdaten ersetzen (aktuell Platzhalter)
- Impressum/Datenschutz-Links im Footer verlinken
- Optional: echte Kundenstimmen/Ergebnisse statt der aktuell rein textuellen Nutzenversprechen ergänzen
